title: "Quickstart"
description: "Start running compliant payroll with Remlo."
---

Whether you are an employer looking to manage your team's salary, an operator configuring an AI agent, or a developer building a custom integration, Remlo provides the tools you need. Choose the path that fits your use case.

### 1. "I want to run payroll from my dashboard"
If you prefer a visual interface, you can manage everything through the Remlo Employer Dashboard.
1. Register your company on the [Remlo App](https://remlo.xyz).
2. Add your employees to your team directory.
3. Deposit funds into your treasury via the Bridge virtual account integration.
4. Run the payroll wizard to execute a compliant batch payout.

### 2. "I want an agent to run payroll autonomously"
Use AgentCash to authorize an AI agent to manage your treasury and execute payroll on your behalf.
1. Set up AgentCash by running `npx agentcash try https://remlo.xyz`.
2. Fund the agent's wallet with the required stablecoin balance for API gas fees.
3. The agent can now securely interact with Remlo by calling the `/api/mpp/agent/session/treasury` endpoint.

### 3. "I want to call the API directly"
Build custom workflows or integrations by calling the MPP API endpoints.
1. Obtain an MPP-compatible wallet to handle standard 402 HTTP payment flows.
2. Review the [MPP API Reference](/docs/mpp-api/overview) and pick the endpoint you need.
3. Send the request handling the 402 payment flow to execute the transaction and retrieve your data.
