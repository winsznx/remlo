Remlo is borderless enterprise payroll on Tempo L1. Employers fund a treasury, run compliant batch payouts, stream salary in real time, issue employee cards through Bridge, and expose the same workflows to software agents through MPP.

## App Surfaces

Remlo provides ready-to-use application surfaces for managing all aspects of the payroll lifecycle:
- **Employer Dashboard**: A comprehensive UI for employers to fund their treasury, run batch payroll, manage team compliance, and monitor yield.
- **Employee Portal**: A dedicated workspace for employees to track their salary, manage their connected wallets, and activate their Visa cards.

## Protocol Layer

Under the hood, Remlo is powered by a set of robust smart contracts deployed on Tempo L1. This protocol layer ensures that all transactions are secure, compliant, and transparent. Every transfer is automatically screened against TIP-403 compliance policies, and payouts are executed efficiently using the Payroll Batcher and TIP-20 memos.

Because the underlying layer is entirely on-chain, you can integrate Remlo's capabilities directly into your own applications or allow your AI agents to interact with the MPP endpoints natively.
